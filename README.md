# Operating-Systems-Coding
A C++ simulation suite covering core Operating System algorithms — CPU scheduling, page replacement, disk scheduling, and memory allocation — with fully interactive input/output for hands-on understanding.
The structure for reference is 
OS-Algorithms/
├── CPU-Scheduling/
│   ├── fcfs.cpp
│   ├── sjf.cpp
│   └── round_robin.cpp
├── Page-Replacement/
│   ├── fifo.cpp
│   └── lru.cpp
├── Disk-Scheduling/
│   ├── sstf.cpp
│   └── scan.cpp
└── Memory-Allocation/
    ├── first_fit.cpp
    └── best_fit.cpp
